# malikdanab.com

Portfolio of **Malik Danab** — Gameplay Programmer & Technical Designer.

**[Visit the site →](https://malikdanab.com)**

Featured work:

- **[InVein](https://malikdanab.com/invein/)** — psychological horror game released on Steam; sound systems, radio system, and rendering work in Unreal Engine 5.
- **[FourSight](https://malikdanab.com/foursight/)** — solo-developed roguelike deck-building tower defense built on Unity DOTS, with flow field pathfinding and a data-driven card keyword system.
- **[FinalPrice](https://malikdanab.com/sales-tax-calculator/)** — Chrome extension published on the Web Store that shows tax-included prices on any shopping site.

Built as a static [Astro](https://astro.build) site and deployed automatically to GitHub Pages on every push to `main`.
